With Frndzk Post From Any Mobile Plugin you can add post to your blog from any mobile browser.

== Description ==

Frndzk Post From Any Mobile Plugin enables you to add post to your blog from any mobile browser. no setup or configuration needed. Just install and you will be able to post from mobile. Plugin Page Link Will be at footer of your admin panel to get easy access. [demo](http://bitto.us/wp/) plugin developed by [bitto kazi](http://bitto.us)

== Installation ==

1. Upload 'frndzk-post-from-any-mobile.zip' to the '/wp-content/plugins/' directory and extract the zip file
2. Activate the plugin through the 'Plugins' menu in WordPress
